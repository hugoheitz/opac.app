opac.app
========

This is the source code for our website at https://opac.app/. The source code for the app itself can be found at https://github.com/opacapp/opacclient.


Dependencies
------------

```
gem install kramdown jekyll sass
```

Build locally
---------------

```
jekyll serve --watch
```
